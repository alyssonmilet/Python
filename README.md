### Tipos de dados

* Int: Números inteiros;
* Str: Uma String é sequência de caracteres.
* Bool: Dados booleanos (True ou False);
* List: Estrutura de dados para armazenamento de conjunto de elementos;
* Dict: Dicionários;
* Files: Arquivos
